Islamic Edu
🧩 Overview
Islamic Edu هو مشروع تعليمي كيهدف لتطوير ألعاب وموارد رقمية لتسهيل تعلم التربية الإسلامية للأطفال والشباب.
كيجمع بين المحتوى التربوي و التجربة التفاعلية عبر ألعاب مبسطة، أنشطة، وتمارين.

🚀 Features
🎮 Interactive Games: ألعاب تعليمية بالرسوميات والأنيميشن.

📚 Educational Content: دروس وأسئلة في العقيدة، العبادات، والسيرة.

🌐 Multilingual Support: العربية، الفرنسية، الإنجليزية، والدارجة المغربية.

🖥️ Responsive Design: واجهة متوافقة مع الحاسوب والهاتف.

🔒 Child Safety Mode: محتوى آمن ومناسب للأطفال.

🛠️ Installation
bash
# Clone the repository
git clone https://github.com/simokimo89/islamic-edu.git

# Navigate to project folder
cd islamic-edu

# Open index.html in your browser
